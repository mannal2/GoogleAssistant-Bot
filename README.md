# GoogleAssistant-Bot
Dialogflow를 활용한 급식봇과 한국사퀴즈봇

## 개발동기
- 급식봇 : 급식메뉴를 빠르게 알기 위해 개발
- 한국사퀴즈봇 : 통학중에 스마트폰을 이용하여 한국사 공부를 하기 위해 개발
